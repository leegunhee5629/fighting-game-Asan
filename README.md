# Bounce Attack (REMASTERED)

2인용 대전 격투 게임 프로토타입입니다. Pygame을 사용하여 개발되었습니다.

## 폴더 구조

```
fighting-game-Asan/
├── assets/                  # 게임 리소스
│   ├── bgm/                  # 배경음악
│   ├── characters/           # 캐릭터 이미지
│   ├── font/                 # 폰트 파일
│   ├── img/                  # 일반 이미지
│   └── maps/                 # 맵 이미지
├── src/                     # 소스 코드
│   ├── main.py               # 메인 게임 루프
│   └── scenes/               # 씬 관리
│       ├── title.py            # 타이틀 화면
│       ├── characters.py       # 캐릭터 선택
│       └── maps.py             # 맵 선택
├── wip/                     # 작업 중인 파일들
├── requirements.txt         # 의존성 목록
└── README.md                # 프로젝트 설명
```
